# @fepack/image

`@fepack/image` provides tools designed to simplify image management. Explore and enhance your image handling with `@fepack/image`

## Features

- load
- checkWebPSupport
- detect
- extractColors

## Installation

```shell
npm install @fepack/image
```

```shell
pnpm add @fepack/image
```

```shell
yarn add @fepack/image
```
